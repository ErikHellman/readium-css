PostCSS

[Implementers’ doc] [WIP]

PostCSS is a tool for transforming CSS with JavaScript. It comes with a vast amount of task-oriented plugins and allows authors to use modern specs which are not implemented yet.

  • PostCSS tutorial
  • PostCSS plugins list

Useful plugins

Here is a list of PostCSS plugins which might prove useful to implementers.

  • Avoid errors: Stylelint
  • Reordering rules and properties: postcss-sorting
  • Adding vendor prefixes: Autoprefixer
  • Use latest CSS syntax: CSS Next
  • Transform CSS variables into a static representation: postcss-css-variables
  • Concatenate stylesheets: postcss-import
